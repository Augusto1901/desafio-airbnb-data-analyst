# Desafío Profesional Data Analyst - Caso Airbnb 🏡📊

Este repositorio contiene el desarrollo del Desafío Profesional para la certificación en Data Analytics de Digital House. El proyecto se centra en el análisis integral de datos del mercado de Airbnb para identificar oportunidades de inversión reales y formular recomendaciones estratégicas.

## 🎯 Objetivo del Proyecto
Realizar un análisis exploratorio descriptivo de los datos de Airbnb, evaluar la rentabilidad por tipo de propiedad y ubicación, y analizar la relación entre los precios y la satisfacción de los usuarios para guiar a posibles inversores.

## 🛠️ Herramientas Utilizadas
* **Lenguaje:** Python
* **Manipulación y Limpieza de Datos:** Pandas, NumPy
* **Visualización:** Matplotlib, Seaborn
* **Presentación Final:** Power BI

## 📂 Estructura del Proyecto
El análisis se divide en tres etapas fundamentales:
1. **Etapa 1: Exploración Visual de los Datos (EDA)** - Análisis inicial para identificar patrones, tendencias y anomalías en los datasets.
2. **Etapa 2: Limpieza y Transformación** - Tratamiento de valores nulos, eliminación de duplicados, gestión de outliers y normalización de variables (ej. precios y fechas).
3. **Etapa 3: Dashboards y Presentación** - Creación de visualizaciones interactivas y storytelling para comunicar los insights de negocio.

## ⚠️ Nota sobre los Datos
Por cuestiones de tamaño y buenas prácticas, los archivos originales del dataset (`listings.csv`, `calendar.csv`, `reviews.csv`) no están incluidos en este repositorio.
